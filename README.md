# OOP_IO-z11_Python

LaserScan (http://docs.ros.org/en/noetic/api/sensor_msgs/html/msg/LaserScan.html) у Range повідомлення (http://docs.ros.org/en/melodic/api/sensor_msgs/html/msg/Range.html).
